SVG container element

g

Grouping with the g element
title/desc

g 요소는 모든 하위 항목을 하나의 그룹으로 그룹화합니다. 일반적으로 해당 그룹에 이름을 부여하는 id 속성이 있습니다. g 요소에 적용한 모든 스타일은 모든 하위 요소에도 적용됩니다. 이를 통해 전체 개체 그룹에 스타일, 변형, 상호 작용 및 애니메이션을 쉽게 추가할 수 있습니다.

use

Reusing elements with the use element

use 요소를 사용하면 그래픽 편집기의 복사-붙여넣기 기능과 유사한 기능을 제공하여 기존 요소를 재사용할 수 있습니다.

defs

Reusing Stored elements with the defs element

defs 요소는 [..] 요소를 직접 렌더링하지 않고 요소를 정의하는 데 사용되며 해당 요소는 향후 사용을 위한 템플릿 역할을 합니다.
